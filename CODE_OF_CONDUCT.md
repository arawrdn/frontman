# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Reporting

If you experience or witness unacceptable behavior, please report it through [GitHub's private reporting](https://github.com/frontman-ai/frontman/security/advisories/new) so that project maintainers can review the matter confidentially.

## Enforcement

Project maintainers are responsible for clarifying and enforcing standards of acceptable behavior and will take appropriate and fair corrective action in response to any behavior that they deem inappropriate.
