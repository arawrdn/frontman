How it works

The split layout widget:

  • Takes over the full page viewport
  • Shows a chat interface on the left panel
  • Displays the original page content in an iframe on the right
  • Built with React and Radix UI icons
  • Injected via Shadow DOM to avoid style conflicts
  • Completely isolated from the host page's CSS

Additional Content
